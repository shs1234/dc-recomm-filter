# Privacy Policy (template)

This extension **does not** collect personal data from users except for settings you explicitly store via the extension's options. The extension stores the following information using Chrome `storage.sync`:
- `enabled` (boolean): whether the recommend-filter is active
- `threshold` (number): the minimum recommend count to show posts

We do not send any data to external servers.

If you later add features that access page content or external services, update this policy accordingly and include a link to the live privacy policy in the store listing.
